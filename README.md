# UI Automation Tests (Pet Project)

Учебный репозиторий для отработки навыков автоматизированного тестирования веб-интерфейсов.

## 🛠 Стек технологий
- **Язык:** Python
- **Фреймворк:** Pytest
- **Библиотека:** Selenium WebDriver
- **Паттерн:** Page Object Model (POM)

## 🏗 Архитектура проекта
Проект разделен на логические слои для удобства поддержки:
- `locators/` — хранение всех локаторов элементов.
- `pages/` — описание страниц и действий с ними (без ассертов).
- `test/` — сами тестовые сценарии с проверками (assert).
- `test_data/` — тестовые данные.
- `conftest.py` — фикстуры для инициализации и закрытия браузера (с флагами для стабильной работы, в т.ч. в CI/CD).

## 🚀 Как запустить
1. Установить зависимости: `pip install -r requirements.txt`
2. Запустить тесты: `pytest`

## 📌 Зоны для развития (Roadmap)
Этот проект демонстрирует понимание базовой архитектуры. Для перевода в продакшн-качество планируется:
- Добавление явных ожиданий (`Explicit Waits` / `WebDriverWait`) вместо неявных.
- Интеграция генерации отчетов через Allure.
- Параметризация тестов через `pytest.mark.parametrize`.
